## Compilar

### Opción 1: GitHub Actions (Recomendado)
1. Sube este código a tu repositorio de GitHub
2. Ve a la pestaña "Actions"
3. Ejecuta el workflow "Build ROM"
4. Descarga el archivo .gba compilado

### Opción 2: Local (Linux/WSL)
```bash
sudo apt-get install build-essential binutils-arm-none-eabi git libpng-dev
make -j$(nproc)
```

## Notas de Traducción
- Sin tildes (á, é, í, ó, ú) por compatibilidad
- ñ reemplazada por n donde sea necesario
- "POKeMON" en lugar de "POKéMON"
- Nombres de especies Pokémon sin traducir (estándar oficial)
- Terminología oficial de Pokémon en español
